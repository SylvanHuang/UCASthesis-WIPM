# UCASthesis-WIPM
a LaTeX template for UCAS-WIPM, 中科院学位论文模板
本模板在UCASthesis(https://github.com/xiaoyao9933/UCASthesis)基础上做了少量修改,适用于中国科学院武汉物理与数学研究所学位论文

修改说明：
ucasthesis.cfg
%中国科学院武汉物理与数学研究所论文原创性声明和使用授权说明

ucasthesis.cls
%格式修改：中英文封面，原创性声明和使用授权说明，致谢，摘要

cover.tex
%单位为中国科学院武汉物理与数学研究所

abs.tex
%增加单独一个摘要文件

main.tex
%将致谢放在摘要之前
